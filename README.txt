Это - приложение для отслеживание количетсва прочитанной человеком литературы, с его кратким отзывом о ней


В нем имеются кнопки добавления, удаления записей о прочитанных книгах

При добавлении записи открывается новая форма, в которой можно заполнить все необходисые данные
Для удаления также надо выделить строку в центральной панели
При нажатии на кнопку "Настройки" открывается форма, в которой можно импортировать или экспортировать таблицу в/из csv-файла
В верхней части таблицы присутсвует кнопка для обновления информации в таблице


Центральная панель представляет из себя QLabel с "натянутым" QTableWidget.

Данные хранятся в базе данных book.sqlite
